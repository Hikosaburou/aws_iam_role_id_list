# aws_iam_role_id_list
IAM Roleと aws:userid を一覧で表示するやつ


# Install

```
pip install -e .
```

# Usage

```
$ iamrolelist
| RoleName              | RoleId                                                           |
|:----------------------|:-----------------------------------------------------------------|
| AROXXXXXXXXXXXXXXXXXX | RoleNameXXXXXX                                                   |
...
```
